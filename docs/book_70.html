User controls

User controls are not ready in the ALPHA version. This feature will be ready in the BETA release. Currently it is possible to create custom controls but the process is very unsafe and limited.

For an example to can check the window template source which is basically custom control. Obviously easier and safer way to create custom controls will be added.
