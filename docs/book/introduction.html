Native Windows GUI: The Guide

Welcome! Native Windows GUI (NWG) is a thin GUI toolkit built over the Windows WINAPI for rust. The current version is 0.2.0 BETA. The library is not production ready, but it has most of its STABLE release functionalities and should be able to create a wide range of GUI applications.
