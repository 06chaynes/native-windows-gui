Button

A rectangle containing an application-defined text label that indicates what the button does when the user selects it. 
pub struct Button<ID: Eq+Clone+Hash> {
    pub text: String,
    pub size: (u32, u32),
    pub position: (i32, i32),
    pub parent: ID,
    pub text_align: (HTextAlign, VTextAlign),
}
Members
  • text: The button label
  • size: The button size in PX
  • position: The button position in its parent
  • parent: The button parent
  • text_align: The button label alignment
Events
  • Click
  • Focus
  • MouseUp
  • MouseDown
  • Removed
Actions
  • SetText: Set the button label text
  • GetText: Return the button label text
  • GetPosition: Return the position of the control
  • SetPosition: Set the widget position
  • GetSize: Return the size of the control
  • SetSize: Set the control size
  • GetParent: Return the id of the parent
  • SetParent: Set the parent of the control
  • GetEnabled: Check if the control is either enabled or not
  • SetEnabled: Disable or enable the control
  • GetVisibility: Check if the control is either visible or not
  • SetVisibility: Show or hide the control
  • GetControlType: Returns constants::ControlType::Button

Checkbox

A square box and an application-defined label that indicates a choice the user can make by selecting the button. Applications typically display check boxes to enable the user to choose one or more options that are not mutually exclusive. 
pub struct CheckBox<ID: Eq+Clone+Hash> {
    pub text: String,
    pub size: (u32, u32),
    pub position: (i32, i32),
    pub parent: ID,
    pub tristate: bool,
    pub text_align: (HTextAlign, VTextAlign),
}
Members
  • text: The checkbox label
  • size: The checkbox size in PX
  • position: The checkbox position in its parent
  • parent: The checkbox parent
  • tristate: If the checkbox should have a third state
  • text_align: The checkbox label alignment
Events
  • Click
  • Focus
  • MouseUp
  • MouseDown
  • Removed
Actions
  • SetText: Set the checkbox label text
  • GetText: Return the checkbox label text
  • GetPosition: Return the position of the control
  • SetPosition: Set the widget position
  • GetSize: Return the size of the control
  • SetSize: Set the control size
  • GetParent: Return the id of the parent
  • SetParent: Set the parent of the control
  • GetEnabled: Check if the control is either enabled or not
  • SetEnabled: Disable or enable the control
  • GetVisibility: Check if the control is either visible or not
  • SetVisibility: Show or hide the control
  • GetControlType: Returns constants::ControlType::CheckBox
  • Reset: Set the check state to unchecked
  • GetCheckState: Get the check state of the control
  • SetCheckState: Set the check state of the control

ComboBox

A control consisting of a dropdown menu and a selection field similar to an edit control. 
pub struct ComboBox<ID: Eq+Clone+Hash> {
    pub size: (u32, u32),
    pub position: (i32, i32),
    pub parent: ID,
    pub collection: Vec<String>,
    pub sorted: bool
}
Members
  • text: The combobox label
  • size: The combobox size in PX
  • position: The combobox position in its parent
  • parent: The combobox parent
  • collection: The items in the combobox dropdown
  • sorted: If the dropdown values should be sorted
Events
  • Click
  • Focus
  • MouseUp
  • MouseDown
  • Removed
  • MenuClose
  • MenuOpen
  • SelectionChanged
  • ValueChanged
Actions
  • GetText: Return the text of the selected item
  • GetPosition: Return the position of the control
  • SetPosition: Set the widget position
  • GetSize: Return the size of the control
  • SetSize: Set the control size
  • GetParent: Return the id of the parent
  • SetParent: Set the parent of the control
  • GetEnabled: Check if the control is either enabled or not
  • SetEnabled: Disable or enable the control
  • GetVisibility: Check if the control is either visible or not
  • SetVisibility: Show or hide the control
  • GetControlType: Returns constants::ControlType::ComboBox
  • Reset: Remove all items from the combobox
  • GetSelectedIndex: Return the index of the selected item
  • SetSelectedIndex: Set the selected item identifier by the its index
  • GetDropdownVisibility: Check if the combobox dropdown is visible
  • SetDropdownVisibility: Hide or show the combobox dropdown
  • AddString: Add an item a the end of the dropdown (unless it is sorted)
  • FindString: Find the index of an item in the dropdown
  • RemoveString: Remove the selected string from the dropdown collection
  • InsertString: Insert an item in the control
  • GetIndexedString: Return the string at the specified index
  • GetStringCollection: Return all the item contained in the dropdown
  • SetStringCollection: Replace the content of the dropdown
  • RemoveIndexedItem: Remove an item at the specified index
  • CountItems: return the number of items in the combobox

Window

A moveable and resizable rectangular area that contains children controls. Has a non-client area with a title and a few options such as closing and minimizing. 
pub struct Window {
    pub caption: String,
    pub size: (u32, u32),
    pub position: (i32, i32),
    pub visible: bool,
    pub resizable: bool,
    pub exit_on_close: bool
}
Members
  • caption: The group box label
  • size: The group box in PX
  • position: The group box position in its parent
  • visible: If the window should be visible from the start
  • resize: If the user should be able to resize the window
  • exit_on_close: If the application should close with the window
Events
  • MouseUp
  • MouseDown
  • Removed
  • Focus
Actions
  • SetText: Set the window title
  • GetText: Get the window title
  • GetPosition: Return the position of the control
  • SetPosition: Set the widget position
  • GetSize: Return the size of the control
  • SetSize: Set the control size
  • GetEnabled: Check if the control is either enabled or not
  • SetEnabled: Disable or enable the control
  • GetVisibility: Check if the control is either visible or not
  • SetVisibility: Show or hide the control
  • GetControlType: Returns constants::ControlType::Window
  • GetWindowDisplay: Get the window state (maxized, minimized, normal)
  • SetWindowDisplay: Set the window state (maxized, minimized, normal)
  • Close: Close the window
